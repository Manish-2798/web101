Introduction to Html


Html(HyperText Markup Language) is the most basic building block of the Web. It defines the meaning and structure of web content. HTML describes the structure of a Web page HTML consists of a series of elements HTML elements tell the browser how to display the content Other technologies besides HTML are generally used to describe a web page's appearance/presentation (CSS) or functionality/behavior (JavaScript).
